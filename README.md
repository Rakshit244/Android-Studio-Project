Overview

The Expense Tracker App is an Android-based application designed to help users efficiently manage their daily income and expenses. It provides a simple, secure, and user-friendly platform for tracking financial activities and improving personal budgeting habits.

This app enables users to record transactions, categorize expenses, and monitor their financial health through clear insights and reports.

*Features:-*
Expense & Income Tracking – Add, edit, and delete financial transactions.
Category-Based Management – Organize expenses (Food, Transport, Bills, etc.).
Visual Reports – View spending patterns using charts and summaries.
Local Data Storage – Securely store user data on the device.
Lightweight & Fast – Optimized for smooth performance on Android devices.
Accessibility Support – Adjustable text size and clean UI for better usability.

*Tech Stack:-*
Language: Java / Kotlin.
Platform: Android Studio.
Database: SQLite / Room Database.
UI Design: XML Layouts.
